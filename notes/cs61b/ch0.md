## 总结

前提:课程网页上说需要做过 CS61A ，或者具有类似课程性质的基础上再来学习 CS61B 。

课程虽然采用 Java ，但是没有 Java 基础也可以直接学习。当然如果有 Java 基础的话会更好！

这是一门讲解数据结构的课程。

* Phase1：Java入门，基础语法，工具简介。
* Phase2：数据结构，将会使用到第一阶段的工具。
* Phase3：算法和软件工程。
